MatchBread Privacy Policy

Effective date: 29/06/2026

MatchBread is a Reddit Devvit app designed to help r/ALeague moderators automatically create and update football match threads.

This Privacy Policy explains what information MatchBread processes, stores, and shares.

1. Information MatchBread does not collect

MatchBread does not intentionally collect, store, sell, or share personal information about Reddit users.

MatchBread does not collect:

Reddit usernames
Reddit user IDs
Email addresses
IP addresses
Private messages
Comments or voting behaviour
User browsing activity
Analytics or advertising identifiers

MatchBread does not use cookies, tracking pixels, advertising tools, or third-party analytics.

2. Information MatchBread processes

MatchBread processes limited information required to operate match threads, including:

Subreddit installation settings configured by moderators
Match fixture IDs
Match status information
Kickoff times
Reddit post IDs for match threads created by the app
Cached football fixture data
Cached live match data
Last fixture refresh time

This information is used only to create, update, and manage football match threads.

3. Moderator-provided settings

Subreddit moderators may configure MatchBread using Reddit’s app settings. These settings may include:

A football API key
An optional weather API key
Target club names
Test mode settings
Optional test fixture IDs
Weather display preferences

These settings are used only for MatchBread functionality.

Moderators should not enter personal information into MatchBread settings unless it is required for their own administration of the app.

4. Third-party services

MatchBread uses external services to fetch football and optional weather data.

API-Football / API-Sports

MatchBread uses v3.football.api-sports.io to retrieve football fixtures, scores, match events, lineups, statistics, and match status information.

Requests to this service may include:

League IDs
Season years
Fixture IDs
Timezone information
The configured football API key

MatchBread does not send Reddit usernames, comments, votes, private messages, or other Reddit user personal information to this service.

WeatherAPI.com

If weather is enabled, MatchBread may use api.weatherapi.com to retrieve current weather information for a match venue city.

Requests to this service may include:

Venue city name
The configured WeatherAPI key

Weather functionality is optional.

5. Data storage

MatchBread stores limited operational data using Reddit Devvit’s Redis storage.

This may include:

Active match thread records
Cached live fixture data
Cached season fixture data
Last fixture refresh timestamp

This data is used only so MatchBread can avoid duplicate posts, continue updating active threads, and reduce unnecessary API calls.

6. Data sharing

MatchBread does not sell user data.

MatchBread only shares limited non-user operational request data with the third-party services required for its functionality, as described above.

MatchBread also posts public match threads and post-match threads to the subreddit where it is installed.

7. Data retention

MatchBread retains operational data only for as long as needed to operate the app.

Cached fixture and live match data may expire automatically. Some operational records, such as active thread references or post-match creation markers, may remain longer to prevent duplicate posts.

Subreddit moderators may remove the app from their subreddit at any time.

8. Children’s privacy

MatchBread is a moderation utility for Reddit communities. It is not directed at children and does not knowingly collect personal information from children.

9. Changes to this Privacy Policy

This Privacy Policy may be updated from time to time. Updates will be published on this page with a revised effective date.
